# Archive - Historical Documentation

이 폴더는 프로젝트 진행 과정에서 생성된 분석 문서들의 아카이브입니다.

## 구조
- **analysis/** - 비판적 분석 및 발견 문서들
- **experiments/** - 실험 결과 및 검증 문서들  
- **deprecated/** - 구버전 가이드 및 중복 문서들
- **old_conclusions/** - 이전 결론들 (Buy & Hold 관련)

## 최신 상태
최신 프로젝트 상태는 상위 폴더의 다음 문서들을 참고하세요:
- PROJECT_STATUS.md
- QUICK_START_GUIDE.md
- README.md
