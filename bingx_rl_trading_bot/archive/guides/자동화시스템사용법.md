## ✅ 성공/실패 기준

### ✅ 성공 신호
```
Dashboard에서:
  - 포트폴리오: +1% 이상
  - 승률: 50% 이상
  - TP 도달률: 20% 이상
  - 경고: 없음
```

### ⚠️ 주의 신호
```
Dashboard에서:
  - 포트폴리오: 0% ~ -1%
  - 승률: 45-50%
  - TP 도달률: 10-20%
  - 경고: 1-2개
```

### ❌ 위험 신호
```
Dashboard에서:
  - 포트폴리오: -2% 이하
  - 승률: 45% 미만
  - TP 도달률: 10% 미만
  - 경고: 3개 이상
```

**위험 신호 시**: 봇 중지 고려 (Ctrl+C 또는 kill)

### Q2. 돈을 잃고 있어요!

**확인**:
```bash
python scripts/production/dashboard.py
```

**판단**:
- -1% 미만: 정상 (변동성)
- -2% ~ -3%: 주의 (모니터링 강화)
- -3% 이상: 위험 (중지 고려)

---

### Q3. TP가 하나도 안 맞아요!

**확인**:
```bash
python scripts/production/monitor_v2_bot.py
```

**판단**:
- 거래 5개 미만: 데이터 부족 (더 기다림)
- 거래 10개, TP 0%: V2도 실패 (V3 필요)
- 거래 20개, TP <10%: 문제 있음 (재검토)

---

### Q4. 승률이 낮아요!

**확인**:
```bash
python scripts/production/dashboard.py
```

**판단**:
- 거래 5개 미만: 데이터 부족
- 거래 10-20개, 승률 45-50%: 수용 가능
- 거래 20개 이상, 승률 <45%: 문제 (개선 필요)
