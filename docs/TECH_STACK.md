# Tech Stack - CLAUDE_CODE_FIN

**Last Updated**: 2025-10-14

## Programming Language
- **Python 3.9+**: 모든 프로젝트의 기본 언어

## Machine Learning / Deep Learning
- **XGBoost**: 현재 프로덕션 모델 (Gradient Boosting)
- **Stable-Baselines3**: 강화학습 프레임워크 (PPO 알고리즘)
- **PyTorch**: 딥러닝 프레임워크 (LSTM 개발 예정)
- **Gymnasium**: RL 환경 구축

## Data Analysis & Visualization
- **pandas**: 데이터 처리 및 분석
- **numpy**: 수치 연산
- **matplotlib**: 데이터 시각화
- **plotly**: 인터랙티브 차트
- **mplfinance**: 금융 차트 (캔들스틱)

## Technical Indicators
- **ta**: 기술적 지표 라이브러리
- **pandas-ta**: 추가 기술적 지표

## API & Trading
- **BingX SDK**: BingX 거래소 API 클라이언트
- **CCXT**: 암호화폐 거래소 통합 라이브러리
- **aiohttp**: 비동기 HTTP 클라이언트
- **websockets**: 실시간 데이터 스트리밍

## Logging & Monitoring
- **loguru**: 구조화된 로깅
- **rich**: 터미널 출력 포맷팅

## Configuration & Environment
- **python-dotenv**: 환경 변수 관리
- **PyYAML**: 설정 파일 관리

## Testing
- **pytest**: 단위 테스트
- **pytest-asyncio**: 비동기 테스트

## Development Tools
- **Git**: 버전 관리
- **VS Code**: 주요 IDE (권장)
